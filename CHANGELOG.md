# Changelog

## [0.1.0] - 2026-04-08

### Added

- Initial release
- `screenshot` tool with region, monitor, output, file_path, and auto_crop parameters
- Cross-platform support: macOS, Linux (X11/XWayland), Windows
- Structured error responses with error codes, valid options, suggestions, and help URLs
- Multi-monitor support
- Auto-crop for trimming uniform borders
- Output modes: base64 (direct image return), file, clipboard
- CI/CD: GitHub Actions for testing (Python 3.11/3.12/3.13) and PyPI publishing
